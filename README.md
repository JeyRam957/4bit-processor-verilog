# Custom 4-Bit Processor in Verilog

## Overview

Designed and simulated a custom 4-bit processor using Verilog HDL.

The processor includes:

- Program Counter
- Instruction Memory
- Control Unit
- Register File
- Arithmetic Logic Unit (ALU)

## Instruction Set

| Opcode | Operation |
|----------|----------|
| 000 | ADD |
| 001 | SUB |
| 010 | AND |
| 011 | OR |
| 100 | MOV |

## Tools Used

- Verilog HDL
- Xilinx Vivado
- Vivado Simulator

## Project Structure

rtl/          -> Verilog source files

testbench/    -> Simulation testbench

images/       -> RTL schematic and waveform

docs/         -> CPU block diagram

## Results

Successfully simulated instruction execution and verified datapath operation using Vivado.
